取色器。

## 何时使用

需要获取颜色色值时。

## API


颜色选择器的属性说明如下：

属性            | 说明                              | 类型    | 默认值
----------------|-----------------------------------|--------|------
defaultValue    | 默认值                             | string | `#ff0000`
mode            | 显示模式，可选值: default \| button  | string | `default`
preset          | 是否开启备选色板                     | boolean | `false`
